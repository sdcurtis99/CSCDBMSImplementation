# Lab 1 – ARIES Recovery Simulation

## Requirements
- Python
- No external libraries required

## Files
- wal.jsonl — write-ahead log
- disk_pages.json — disk state at crash
- recovery.py — ARIES recovery simulator

## How to Run
Place `wal.jsonl` and `disk_pages.json` in the same directory as `ARIESRecoverySim.py`.
Use absolute paths to alter file names to check test case.

Run:
```bash
python ARIESRecoverySim.py
